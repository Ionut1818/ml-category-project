# Product Category Classification System

This is a Machine Learning System that automatically classifies the products in categories based on their names, it uses the LinearSVC algorhythm.

# Characteristics:

Advanced model: LinearSVC with TF-IDF vectorization

Big data: Over 1000 products

Real-time prediciton system

High Performance: F1 score optimized for classification
